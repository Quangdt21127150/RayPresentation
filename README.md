# Ray Presentation

## Hướng dẫn chạy code (Ray 2.54.0, Python 3.10 - 3.12)

1. Cài đặt framework/thư viện:
   pip install -r requirements.txt

2. Chạy các ví dụ:
   - python 01\_{...}.py
   - python 02_architecture_example.py
   - python 03_ray_serve\_{...}.py

3. Ray Serve:
   - Chạy file 03_ray_serve\_{...}.py
   - Chạy Postman URL: http://127.0.0.1:8000/{...}
   - Để dừng Serve: Ctrl + C

Tất cả code đều chạy được trên máy local (single node).
